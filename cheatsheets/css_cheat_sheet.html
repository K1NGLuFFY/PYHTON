CSS Selectors

/* Element Selector */ p { color: blue; } /* Class Selector */ .highlight { background: yellow; } /* ID Selector */ #header { font-size: 24px; } /* Attribute Selector */ input[type="text"] { border: 1px solid #ccc; } /* Pseudo-classes */ a:hover { color: red; } a:active { color: green; } /* Pseudo-elements */ p::first-line { font-weight: bold; } p::before { content: "â†’ "; } /* Combinators */ div p { } /* Descendant */ div > p { } /* Child */ h1 + p { } /* Adjacent sibling */ h1 ~ p { } /* General sibling */

Transitions & Animations

/* Transitions */ transition: all 0.3s ease; transition: background-color 0.5s ease-in-out; /* Animations */ @keyframes slideIn { from { transform: translateX(-100%); } to { transform: translateX(0); } } .element { animation: slideIn 0.5s ease-out; animation-delay: 1s; animation-iteration-count: infinite; }

Responsive Design

/* Media Queries */ @media (max-width: 768px) { .container { flex-direction: column; } } @media (min-width: 769px) and (max-width: 1024px) { .container { grid-template-columns: 1fr 1fr; } } /* Responsive Units */ width: 100vw; /* viewport width */ height: 100vh; /* viewport height */ font-size: 2vw; /* viewport width unit */
Breakpoint Device Width
Mobile Phones < 768px
Tablet iPads 768px - 1024px
Desktop Laptops > 1024px

CSS Variables (Custom Properties)

:root { --primary-color: #667eea; --secondary-color: #764ba2; --font-size: 16px; --spacing: 1rem; } .element { color: var(--primary-color); font-size: var(--font-size); padding: var(--spacing); }
Benefits: Easy theme switching and consistent design system

Modern CSS Features

/* CSS Grid with minmax */ grid-template-columns: repeat(auto-fit, minmax(250px, 1fr)); /* Clamp for responsive sizing */ font-size: clamp(1rem, 2vw, 2rem); /* Aspect Ratio */ aspect-ratio: 16 / 9; /* Backdrop Filter */ backdrop-filter: blur(10px); /* Scroll Snap */ scroll-snap-type: x mandatory; scroll-snap-align: start;

Common Utility Classes

/* Centering */ .center { display: flex; justify-content: center; align-items: center; } /* Text Utilities */ .text-center { text-align: center; } .text-truncate { overflow: hidden; text-overflow: ellipsis; white-space: nowrap; } /* Spacing Utilities */ .m-1 { margin: 0.25rem; } .p-2 { padding: 0.5rem; } .mb-3 { margin-bottom: 1rem; } /* Display Utilities */ .d-none { display: none; } .d-block { display: block; } .d-flex { display: flex; }
