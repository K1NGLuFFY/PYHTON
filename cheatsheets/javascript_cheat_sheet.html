JavaScript Cheat Sheet

Quick reference for modern JavaScript development

Variables & Data Types

Variable Declarations

// ES6+ declarations let x = 5; // Mutable variable const y = 10; // Immutable constant var z = 15; // Function-scoped (avoid in modern JS) // Data Types const string = "Hello World"; const number = 42; const boolean = true; const array = [1, 2, 3, 4, 5]; const object = { name: "John", age: 30 }; const nullValue = null; const undefinedValue = undefined; const symbol = Symbol('unique');

Type Checking

typeof "hello" // "string" typeof 42 // "number" typeof true // "boolean" typeof [] // "object" typeof null // "object" (bug!) Array.isArray([]) // true null === null // true

Arrays

Array Methods

  • push() - Add to end
  • pop() - Remove from end
  • shift() - Remove from start
  • unshift() - Add to start
  • map() - Transform elements
  • filter() - Filter elements
  • reduce() - Reduce to single value
  • find() - Find first match
  • includes() - Check if contains
  • slice() - Extract portion
  • splice() - Add/remove elements

Array Examples

const numbers = [1, 2, 3, 4, 5]; // Map const doubled = numbers.map(n => n * 2); // [2, 4, 6, 8, 10] // Filter const evens = numbers.filter(n => n % 2 === 0); // [2, 4] // Reduce const sum = numbers.reduce((acc, n) => acc + n, 0); // 15 // Find const found = numbers.find(n => n > 3); // 4

Objects

Object Operations

// Object creation const person = { name: "Alice", age: 25, greet() { return `Hello, I'm ${this.name}`; } }; // Access properties person.name; // "Alice" person['age']; // 25 // Add/Update properties person.email = "alice@email.com"; person.age = 26; // Object methods Object.keys(person); // ["name", "age", "email", "greet"] Object.values(person); // ["Alice", 26, "alice@email.com", function] Object.entries(person); // [[key, value] pairs] // Destructuring const { name, age } = person; const { name: userName, age: userAge } = person;

Functions

Function Types

// Function declaration function greet(name) { return `Hello, ${name}!`; } // Function expression const greet = function(name) { return `Hello, ${name}!`; }; // Arrow function (ES6) const greet = (name) => `Hello, ${name}!`; const add = (a, b) => a + b; // Default parameters const greet = (name = "Guest") => `Hello, ${name}!`; // Rest parameters const sum = (...numbers) => numbers.reduce((a, b) => a + b, 0); // Higher-order functions const createMultiplier = (multiplier) => (number) => number * multiplier; const double = createMultiplier(2); double(5); // 10

ES6+ Features

Modern JavaScript

// Template literals const name = "World"; const greeting = `Hello, ${name}!`; // Spread operator const arr1 = [1, 2, 3]; const arr2 = [...arr1, 4, 5]; // [1, 2, 3, 4, 5] // Object spread const obj1 = { a: 1, b: 2 }; const obj2 = { ...obj1, c: 3 }; // { a: 1, b: 2, c: 3 } // Optional chaining const user = { profile: { name: "John" } }; const userName = user?.profile?.name; // "John" const userAge = user?.profile?.age; // undefined // Nullish coalescing const value = null ?? "default"; // "default" const count = 0 ?? 42; // 0 (not 42!) // Destructuring with defaults const { x = 0, y = 0 } = { x: 5 };

Async JavaScript

Promises & Async/Await

// Promise const fetchData = () => { return new Promise((resolve, reject) => { setTimeout(() => resolve("Data loaded"), 1000); }); }; // Using Promise fetchData() .then(data => console.log(data)) .catch(error => console.error(error)); // Async/Await async function loadData() { try { const data = await fetchData(); console.log(data); } catch (error) { console.error(error); } } // Multiple async operations async function loadMultiple() { const [users, posts] = await Promise.all([ fetch('/api/users'), fetch('/api/posts') ]); return { users, posts }; }

Common DOM Methods

// Select elements const element = document.getElementById('myId'); const elements = document.querySelectorAll('.myClass'); const firstElement = document.querySelector('.myClass'); // Create elements const newDiv = document.createElement('div'); newDiv.textContent = 'Hello World'; newDiv.className = 'my-class'; // Modify elements element.textContent = 'New text'; element.innerHTML = 'Bold text'; element.style.color = 'blue'; element.classList.add('active'); element.classList.remove('inactive'); element.classList.toggle('visible'); // Event listeners element.addEventListener('click', (event) => { console.log('Element clicked!', event.target); }); // Remove elements element.remove(); parentElement.removeChild(childElement);

String Methods

Common String Operations

const str = "Hello, World!"; // Basic methods str.length; // 13 str.toLowerCase(); // "hello, world!" str.toUpperCase(); // "HELLO, WORLD!" str.trim(); // Remove whitespace str.trimStart(); // Remove leading whitespace str.trimEnd(); // Remove trailing whitespace // Searching str.includes("World"); // true str.startsWith("Hello"); // true str.endsWith("!"); // true str.indexOf("o"); // 4 str.lastIndexOf("o"); // 8 // Extracting str.slice(0, 5); // "Hello" str.substring(0, 5); // "Hello" str.substr(7, 5); // "World" str.split(", "); // ["Hello", "World!"] // Replacing str.replace("World", "JavaScript"); // "Hello, JavaScript!" str.replaceAll("l", "L"); // "HeLLo, WorLd!"

Date & Time

Working with Dates

// Create dates const now = new Date(); const specific = new Date('2024-01-15'); const timestamp = new Date(1704067200000); // Get date components now.getFullYear(); // 2024 now.getMonth(); // 0-11 (January is 0) now.getDate(); // 1-31 now.getDay(); // 0-6 (Sunday is 0) now.getHours(); // 0-23 now.getMinutes(); // 0-59 now.getSeconds(); // 0-59 now.getMilliseconds(); // 0-999 // Formatting now.toISOString(); // "2024-01-15T12:00:00.000Z" now.toLocaleDateString(); // "1/15/2024" now.toLocaleTimeString(); // "12:00:00 PM" // Date calculations const tomorrow = new Date(); tomorrow.setDate(tomorrow.getDate() + 1); const diff = date2.getTime() - date1.getTime(); // milliseconds const days = diff / (1000 * 60 * 60 * 24); // days

Error Handling

Try/Catch & Error Types

// Basic try/catch try { // Code that might throw const result = riskyOperation(); } catch (error) { console.error('Error:', error.message); } finally { // Always runs console.log('Cleanup'); } // Custom errors class ValidationError extends Error { constructor(message) { super(message); this.name = "ValidationError"; } } // Throwing errors function validateAge(age) { if (age < 0) { throw new ValidationError("Age cannot be negative"); } return true; } // Error boundaries (async) async function fetchWithError() { try { const response = await fetch('/api/data'); if (!response.ok) { throw new Error(`HTTP ${response.status}`); } return await response.json(); } catch (error) { console.error('Fetch failed:', error); throw error; } }
